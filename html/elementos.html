JavaScript HTML DOM Elements

A continuación se muestra cómo buscar y acceder a los elementos HTML de página web.

Encontrar elementos HTML

Existen varias formas de encontrar los elementos HTML:

  • Encontrar elementos HTML por ID
  • Encontrar elementos HTML por nombre de etiqueta
  • Encontrar elementos HTML por nombre de clase
  • Encontrar elementos HTML mediante selectores CSS
  • Encontrar elementos HTML por colecciones de objetos HTML

Encontrar elementos HTML por ID

Encuentra el elemento con id="ejemplo1".

Sintaxis

const element = document.getElementById("ejemplo1" );

document.getElementById("texto" ).innerHTML = element.innerHTML;

Encontrar elementos HTML por nombre de clase

Para encontrar todos los elementos HTML con el mismo nombre de clase, se utiliza getElementsByClassName().

Encuentra una lista de todos los elementos con class="ejemplo3".

Sintaxis

const element3 = document.getElementsByClassName("ejemplo3" );

document.getElementById("texto3" ).innerHTML = element3[1].innerHTML;

Encontrar elementos HTML mediante selectores CSS

Para encontrar todos los elementos HTML que coincidan con un selector de CSS especificado (id, nombres de clase, tipos, atributos, valores de atributos, etc.), se utiliza querySelectorAll().

Encuentra una lista de todos los <p> elementos con class="ejemplo4".

Sintaxis

const element4 = document.querySelectorAll("p.ejemplo4" );

document.getElementById("texto4" ).innerHTML = element4[0].innerHTML;
