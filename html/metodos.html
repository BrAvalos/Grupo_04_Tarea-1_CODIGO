  • JS HTML DOM - BÁSICO

Se puede acceder al DOM HTML con JavaScript (y con otros lenguajes de programación). En el DOM, todos los elementos HTML se definen como objetos .

Métodos en HTML DOM

Los métodos HTML DOM son acciones que puede realizar (en elementos HTML).

Propiedades en HTML DOM

Las propiedades HTML DOM son valores (de elementos HTML) que puede establecer o cambiar.

Ejemplo:

document.getElementById("demo").innerHTML = "Hello World!";

Donde: getElementById: es un método e innerHTML: es una propiedad

El método getElementById

La forma más común de acceder a un elemento HTML es utilizar el iddel elemento.

En el ejemplo anterior, el getElementByIdmétodo utilizado id="demo"para encontrar el elemento.

La propiedad innerHTML

La forma más sencilla de obtener el contenido de un elemento es mediante la innerHTMLpropiedad.

La innerHTMLpropiedad es útil para obtener o reemplazar el contenido de elementos HTML.
